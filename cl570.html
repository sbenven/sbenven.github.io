Colony Counter

CL-570

Suitable for counting bacteria growing on all agar contained in different size of petri dishes.
The Colony Counter CL-570 is an easy-to-use manual counting device. The pressure sensor detects each count as the operator marks each colony with a pen, and the LED readout displays the count. This device is suitable for counting bacteria growing on all agar contained in different sizes of Petri dishes.

Features
 Specifications Inquiry

Features

  • A reset button - A reset button provides immdiate zeroing of the 4-digit counter.
  • Adjustable focusing arm - Adjustable focusing arm allows 1.5x standard lens to be raised or lowered up to youer preference.
  • Blight fluorescent light - A blight, ring type fluorescent light for good visibility.
  • Red LED display - Easy-to-read red LED displays 0000 to 9999 counts.
  • Sound confirmation - The count is done by marking to the Petri dish and confirmed with the electric sound every 1 count.
